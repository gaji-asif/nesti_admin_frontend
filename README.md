# Nesti Admin Frontend

A React-based admin dashboard built with Tailwind CSS.

## Basic Functions

- **Forms**: Manage form elements and inputs.
- **Authentication**: Handle user sign-in and sign-up processes.
- **Service Management**: Add new services through NewServiceForm component.
- **Category Management**: Add new categories through NewCategoryForm component.

## Getting Started

1. Install dependencies: `npm install`
2. Run the development server: `npm run dev`
3. Build for production: `npm run build`
4. Run tests: `npm test`

## Technologies Used

- React
- TypeScript
- Tailwind CSS
- React Router
- Vitest (for testing)
